# Beam Structure 3D Viewer

Project visualisasi struktur beam 3D untuk monitoring inspeksi.

## Cara Menjalankan

1. Clone repository
git clone https://github.com/dayinta14/beam-structure-3d.git

2. Masuk folder project
cd beam-structure-3d

3. Install dependency
npm install

4. Jalankan server
node server.js

5. Buka di browser
http://localhost:3000
